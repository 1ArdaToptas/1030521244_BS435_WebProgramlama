## Proje Açıklaması
Bu proje, kullanıcının üç görsel arasından hangisinin yapay zeka (AI) tarafından
üretilmiş olduğunu tahmin etmeye çalıştığı etkileşimli bir web oyunudur.
Proje React (Vite) kullanılarak geliştirilmiştir ve tamamen fonksiyonel
component yapısı kullanılmıştır.

---

## Oyun Akışı
1. Oyun modu seçilir (Kolay / Zor)
2. Oyuncuya 3 görsel gösterilir
3. Oyuncu AI olduğunu düşündüğü görseli seçer
4. Yanlış seçim yapılırsa ipucu verilir (Kolay mod)
5. Oyuncuya ikinci bir seçim hakkı tanınır
6. Süre dolarsa oyun otomatik olarak biter
7. Skor hesaplanır ve liderlik tablosuna eklenir

---

## Oyun Modları
- **Kolay Mod:** İpucu verilir, süre daha uzundur
- **Zor Mod:** İpucu verilmez, süre daha kısadır
